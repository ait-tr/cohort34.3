##### Web testing


**What needs to be tested?**


* Functionality
* Usability
* Interface
* Database
* Compatibility
* Performance
* Security


**What is special about web testing?**


* Client-server architecture
* Millions of users(different people with different intentions, uncontrolled)
* Roles and levels
* Code is partle visible
* Cross-platform
